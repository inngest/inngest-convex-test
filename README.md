# inngest-convex

A test repo to prototype and test a Convex adapter for Inngest functions.

**NOTE** - This requires Inngest TypeScript SDK 3.40.0 or later.

```
pnpm i
pnpx convex dev
```

Run Inngest dev server with the actions URL:

```
pnpx inngest-cli@latest dev -u http://localhost:3211/api/inngest
```
